Sharik Game

Sharik Game — это аркадная игра на Python с использованием библиотеки Pygame. Игрок управляет шариком, уклоняется от падающих препятствий, собирает бонусы и зарабатывает очки. С увеличением времени игра ускоряется, и сложность возрастает.

Описание геймплея

Игрок управляет шариком, перемещаясь по горизонтали.

Сверху падают блоки трёх типов:

обычные,

быстрые,

замедленные (тяжёлые).

В игре присутствуют бонусы:

Score — добавляет очки,

Slow — временно замедляет падение блоков,

Shield — делает игрока временно неуязвимым.

У игрока три жизни, которые отображаются в виде красных сердечек.

При столкновении:

уменьшается количество жизней,

включается визуальный эффект тряски экрана и красного мигания.

При потере всех жизней игра заканчивается.

Лучший результат сохраняется в файле sharik_record.txt.

Управление

Стрелка влево — движение влево

Стрелка вправо — движение вправо

SPACE — начать игру из меню

P — поставить игру на паузу

W — продолжить игру

R — перезапустить после окончания

Q — выход из игры

Требования

Python 3.10 или новее

Установленный модуль pygame

Все зависимости указаны в requirements.txt.

Установка и запуск

Склонируйте репозиторий:

git clone https://github.com/xasauzzz/sharik-game.git
cd sharik-game


или скачайте ZIP через кнопку Code → Download ZIP.

Установите зависимости:

pip install -r requirements.txt


Запустите игру:

python sharik.py
